# Machine-Learning-zur-Optimierung-von-Marketingstrategien-fuer-Festgeldanlagen
Im Fokus dieses Projekts steht die Identifikation abschlussstarker Zielgruppen für Festgeldanlagen durch den Einsatz von Machine Learning. Um die Effizienz zukünftiger Marketingkampagnen zu steigern, transformiere ich demographische Kundendaten mittels One-Hot- und Dummy-Kodierung und trainiere verschiedene Klassifikationsmodelle. Ein wesentlicher Bestandteil der Arbeit ist die Validierung der Vorhersagequalität durch Cross-Validated ROC, um sicherzustellen, dass das Modell auch auf unbekannte Datenbestände robust reagiert (AUC: 0,6252). Die Analyse arbeitet heraus, welche Kundensegmente – unterteilt nach Alter und Berufsfeldern – eine geringere Abschlussneigung zeigen. Diese Erkenntnisse dienen als direkte Entscheidungsgrundlage für eine datenbasierte und ressourceneffiziente Steuerung von Vertriebsmaßnahmen.
